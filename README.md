# Karo Pitch - Landing Page

Welcome to the Karo Pitch landing page prototype. This project is a modern, responsive, and aesthetically pleasing landing page designed to showcase a startup pitching platform aimed at founders from Bharat (Tier-2 and Tier-3 cities in India).

## Live Demo (Local)
To view this project locally, simply open the index.html file in any modern web browser, or serve it using a local development server (like VS Code Live Server or Python's http.server).

## Features
*   Modern UI: Features a dark theme with vibrant neon gradients and elegant glassmorphism effects.
*   Fully Responsive: The layout adapts seamlessly to desktops, tablets, and mobile devices.
*   Smooth Animations:
    *   Floating shapes in the hero section.
    *   Scroll-triggered fade-in animations using the IntersectionObserver API.
    *   Smooth scrolling navigation.
*   Sticky Navbar: The navigation bar transforms nicely upon scrolling for better user experience.

## Technologies Used
This project is built using core web technologies without heavy frameworks:
*   HTML5: Semantic structure.
*   CSS3: Custom properties (variables), Grid, Flexbox, Animations, and Backdrop-filters for the glass effect.
*   Vanilla JavaScript (ES6+): DOM manipulation, event listeners, and scroll observers.
*   Fonts & Icons: Google Fonts (Outfit & Inter) and Font Awesome 6.

## Project Structure
*   index.html - The main structure of the landing page.
*   style.css - All styling, variables, and responsive media queries.
*   script.js - Interactivity, scroll animations, and mobile menu logic.

## Sections Included
1.  Hero Section: High-impact headline with floating background elements.
2.  About: "Bridging the Gap for Bharat Founders" highlighting the mission.
3.  How It Works: A 4-step process guide.
4.  Who Can Apply: Categories of target startups.
5.  Investors: Prominent investor placeholder logos.
6.  Featured Startups: Example profiles using a card layout.
7.  Footer: Contact information and social links.
